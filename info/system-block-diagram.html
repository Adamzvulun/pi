Proposal-style block diagram — as built

Same five-layer structure as the original proposal: user → CPU → driver chips → actuators/sensor. Updated to reflect the current build:

  • Camera is now a USB LifeCam HD-3000, not a Pi CSI camera (the v3 was incompatible with the Pi 4 slot).
  • Laser drive path has no MOSFET — GPIO18 directly drives the 3 V self-contained module.
  • Labels show the protocol on each link.

The two power rails are intentionally separated: the Pi USB-C feeds the Pi and the PCA9685's logic supply (~50 mA); the LM2596 feeds the PCA9685's servo power rail (up to 3 A). Common ground across all three. This isolation prevents a servo current spike (up to 2 A at stall) from browning out the Pi.

    KEY --> GATE --> GPIOOUT --> BEAM

The horizontal arrow at the bottom (AIM → SCENE) is the closed loop — moving the bracket changes what the camera sees, which closes the feedback. The right-hand chain is the orthogonal laser-fire path that runs only when the gate conditions are met.

What changed since the original proposal

For comparison — the differences between the proposal-stage design and what actually got built.

BlockProposalAs builtReason
Camera Pi Camera v3 over CSI ribbon LifeCam HD-3000 over USB The v3 wasn't compatible with the Pi 4's 15-pin CSI slot
Laser drive GPIO → IRLZ44N MOSFET → bare 5 mW diode + 100 Ω + 100 kΩ pulldown GPIO18 → 3 V module direct (no MOSFET, no resistor) Replacement laser arrived as an integrated 3 V module with its own driver — MOSFET became redundant
Servo power MB102 breadboard PSU LM2596 buck off 12 V PSU MB102 maxes at ~700 mA; DS3225 needs more (problem 001)
PCA9685 logic supply MB102 Pi GPIO 5 V pin One fewer power module in the chain after MB102 removal
Detection HSV thresholding Same — HSV (blue target) —
Control PID per axis Same — P-only (Ki = Kd = 0), with coast & recenter extensions Empirical tuning showed integral & derivative weren't useful
Operator UI Not in proposal tkinter control panel Added for safety + tooling — replaces ad-hoc terminal commands
