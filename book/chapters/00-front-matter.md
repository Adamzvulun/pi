## תקציר הפרויקט

הפרויקט מציג מערכת אוטונומית מלאה לזיהוי, עקיבה וכיוון לייזר בזמן אמת, הבנויה
סביב מחשב לוח Raspberry Pi 4B. המערכת מצלמת את הסביבה במצלמת USB, מאתרת מטרה
צבעונית בעזרת אלגוריתם זיהוי במרחב צבע HSV, מחשבת את סטיית המטרה ממרכז התמונה
בפיקסלים, ומפעילה לולאת בקרה סגורה (PID) על שני סרוו־מנועים דיגיטליים מסוג
DS3225 המסובבים מערך פאן־טילט דו־צירי. בסיום פעולת ההתכנסות — כאשר המטרה
ממוקמת במרכז התמונה בתוך סף סבילות של 15 פיקסלים בשני הצירים — המערכת
ממתינה לאישור המפעיל ויורה אלומת לייזר אדומה (5 מ"ו, 650 ננומטר) אל המטרה
לפרק זמן קצוב.

הליבה האלגוריתמית כוללת זיהוי מבוסס סף HSV עם פעולות מורפולוגיות
(`erode`/`dilate`) לסינון רעש, חישוב מרכז מסה (centroid) של הקונטור הגדול
ביותר, ובקרת PID עצמאית לכל ציר עם הגנות תכנותיות נגד הפעלה מחוץ לגבולות
המכניים שנקבעו במהלך כיול. המערכת תומכת גם ב־**Coast Mode** (המשך תנועה
אינרציאלי של ~1 שניה לאחר אובדן מטרה) וב־**Recenter Mode** (חזרה חלקה למצב
ההתחלה כאשר ה־Coast נכשל) — שני מנגנונים שנוספו לאחר שתאוצת תנועה מהירה של
המטרה גרמה למערכת להישאר תקועה ב"פינה" של טווח הסריקה.

הפרויקט מדגים אינטגרציה מלאה בין שלוש דיסציפלינות הנדסיות: **חומרה** (תכנון
ספק כוח חצוי עם מייצב 5V מבוסס LM2596 לסרוו ו־Pi GPIO לחיישנים, וחשמלית רגישות
ל־I²C ולמיתוג MOSFET), **תוכנה** (ארכיטקטורת מודולי בעלות־בלעדית עם
`config.py` כמקור־אמת יחיד לקבועים הניתנים לכיול), **ובקרה** (כיול PID
אמפירי מבוסס תצפית, ועיצוב מערכת היגב עם משוב בזמן אמת ממצלמה ל־30 fps).

הפרויקט בנוי על קוד פתוח, פותח בסביבת פיתוח־הפצה תלת־מכונית (מחשב נייד →
GitHub → Raspberry Pi עם משיכה אוטומטית כל דקה), ומגיע עם ממשק־מפעיל גרפי
(`control_panel.py`) המאפשר ביצוע כל הבדיקות החומרתיות, הכיולים, ומפקח על
המערכת ללא צורך בפקודות שורה.

## מילון מונחים קצר

| מונח עברי | מונח אנגלי | פירוש |
|---|---|---|
| לולאת בקרה סגורה | closed-loop control | מערכת שבה הפלט (זווית הסרוו) משפיע על הקלט (סטיית המטרה בתמונה) דרך משוב |
| בקרת PID | PID control | בקר עם רכיב פרופורציונלי (P), אינטגרלי (I) ונגזרתי (D) |
| מרכז מסה / מרכז גיאומטרי | centroid | הנקודה הממוצעת של פיקסלי המטרה המסומנים במסכה |
| מסכה בינארית | binary mask | תמונה שחור־לבן שבה לבן מסמן "פיקסל שייך למטרה" |
| מרחב צבע HSV | HSV color space | ייצוג צבע בעזרת גוון (H), רוויה (S) ובהירות (V) במקום RGB |
| סרוו דיגיטלי | digital servo | סרוו עם בקר אלקטרוני פנימי הפועל ב־PWM ברזולוציה גבוהה |
| גוף MOSFET | MOSFET | טרנזיסטור־שדה לסוויצ'ינג מהיר, משמש כאן כמתג ללייזר |
| באק־קונברטר | buck converter | מייצב מתח שטף־ירידה (LM2596 כאן) |
| Coast Mode | Coast Mode | מצב המשך תנועה אינרציאלי לאחר אובדן זיהוי המטרה |
| Recenter Mode | Recenter Mode | מצב חזרה חלקה למיקום המוצא לאחר כשל ב־Coast |
| Boresight | Boresight | היסט זוויתי בין ציר המצלמה לציר הלייזר, מתוקן בתוכנה |

## תודות

תודה למנחה משה זזק על ההכוונה לאורך כל שלבי הפרויקט; לקהילת ה־Raspberry Pi
על תיעוד הספריות והדוגמאות הפתוחות; וליוצרי הספריות `simple-pid`,
`adafruit-circuitpython-servokit`, `OpenCV` ו־`gpiozero` שעליהן נשענת
המערכת הזו.
